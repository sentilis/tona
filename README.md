## Getting started

### Install using pip

```bash
$ pip install -U tona
$ tona webapp
$ Running on http://0.0.0.0:5001/ (Press CTRL+C to quit
```

### Install using docker-compose

```bash
$ docker-compose up
$ Running on http://0.0.0.0:5001/ (Press CTRL+C to quit
```
